# 🎨 UI 布局优化方案

## 📊 当前问题分析

### 1. 间距系统混乱
**问题**:
- padding: `6px 8px`, `6px`, `8px` 混用
- gap: `6px`, `8px`, `4px` 不统一
- margin: 随意使用，没有规律

**影响**:
- 视觉上参差不齐
- 缺少呼吸感
- 难以维护

### 2. 字体系统不规范
**问题**:
- 字号: `9px`, `10px`, `11px`, `12px` 混用
- 字重: `500`, `600`, `700` 随意
- 行高: 硬编码或缺失

**影响**:
- 视觉层级不清晰
- 阅读体验差
- 缺少统一感

### 3. 颜色使用混乱
**问题**:
- 227 处硬编码颜色
- 透明度值随意：`0.95`, `0.92`, `0.78` 等
- 缺少语义化命名

**影响**:
- 视觉不统一
- 难以维护主题
- 对比度不够

---

## 🎯 优化原则

### 1. 8px 网格系统
所有间距必须是 8 的倍数：
- 4px (0.5x) - 极小间距
- 8px (1x) - 基础单位
- 12px (1.5x) - 小间距
- 16px (2x) - 标准间距
- 24px (3x) - 中等间距
- 32px (4x) - 大间距

### 2. 字体层级系统
```
标题层级:
- H1: 24px / 700 / 1.2
- H2: 20px / 600 / 1.3
- H3: 16px / 600 / 1.4

正文层级:
- Body Large: 14px / 400 / 1.5
- Body: 12px / 400 / 1.5
- Body Small: 11px / 400 / 1.5
- Caption: 10px / 400 / 1.4
```

### 3. 颜色对比度
```
文本颜色:
- Primary: rgba(255, 255, 255, 0.95) - 主要文本
- Secondary: rgba(255, 255, 255, 0.70) - 次要文本
- Tertiary: rgba(255, 255, 255, 0.50) - 辅助文本
- Disabled: rgba(255, 255, 255, 0.30) - 禁用文本

背景颜色:
- Canvas: #0a0a0b - 画布背景
- Surface: rgba(15, 15, 18, 0.95) - 表面
- Elevated: rgba(20, 20, 25, 0.98) - 悬浮
```

## 🔧 具体优化方案

### 阶段 1：统一间距系统

**ControlPanel.tsx 优化**:
```typescript
// 修改前
padding: '6px 8px 6px 8px'  // ❌ 不规范
gap: '6px'                   // ❌ 不是 8 的倍数

// 修改后
padding: 'var(--p-space-2)'  // ✅ 8px
gap: 'var(--p-space-2)'      // ✅ 8px
```

**Tab 组件优化**:
```typescript
// 修改前
height: '30px'               // ❌ 不是 8 的倍数
padding: '6px'               // ❌ 不规范

// 修改后
height: 'var(--p-space-8)'   // ✅ 32px
padding: 'var(--p-space-2)'  // ✅ 8px
```
